# swanton
Swanton Pacific Ranch chatbot with a knowledge graph on a Raspberry Pi

## Getting Started

### On a fresh Debian/Raspbian image
```
sudo apt update \
    && sudo apt install -y git \
    && git clone https://github.com/calpoly-csai/swanton \
    && cd swanton \
    && ./debian_setup.sh
```
